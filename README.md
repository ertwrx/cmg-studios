# CMG Studio — Landing Page

A static landing page for **CMG Studio**, a personal handmade business offering custom shadow boxes, tote bags, toddler activity workbooks, bookmarks, and stickers/labels.

**Live site:** _https://ertwrx.github.io/cmg-studios/_

---

## About This Project

This is the landing page for my personal business, CMG Studio. Beyond serving the business itself, building this site is part of my own learning journey toward becoming a DevOps engineer — working through real decisions around frontend structure, design systems, static hosting, DNS, and deployment instead of just reading about them.

## Tech Stack

- HTML5 / CSS3 (no build step, no framework)
- Google Fonts: Poppins (headings) + Inter (body)
- Deployed via GitHub Pages

## Running Locally

This is a single static HTML file with no dependencies or build step.

```bash
git clone https://github.com/ertwrx/cmg-studios.git
cd cmg-studios
```

Then just open `index.html` directly in a browser, or serve it locally:

```bash
python3 -m http.server 8000
```

Visit `http://localhost:8000`.

## Deployment
 
This site is deployed via **GitHub Pages**, set to deploy automatically from the main branch. Any push to `main` triggers a redeploy — check the **Actions** tab on GitHub to confirm a deployment succeeded.
